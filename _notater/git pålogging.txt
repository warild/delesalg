---

1. logg inn på ssh server 
ssh lastebildemontering.no@ssh.lastebildemontering.no

2. opprett  --bare repo på server
git init --bare deler.git

3. Lokalt: lag lokal Clone fra sentralt repo: 
git clone ssh://lastebildemontering.no@ssh.lastebildemontering.no/~/deler.git deler

---

4. Navnepeker til sentralt repo
gå inn i mappen:
git remote add deler ssh://lastebildemontering.no@ssh.lastebildemontering.no/
~/deler.git

----
Følgende er ikke nødvendig - sentral repo heter 'origin' by default.
Bytte navnepeker: 'origin' = server. 'master' = lokalt

git push --set-upstream origin master
----
Bytte navn på lokal branch? Clone? 

git pull = git fetch + git merge
FETCH_HEAD = peker til det som nettopp er utsjekket fra remote repo 
HEAD = lokal peker til utsjekket working dir. HEAD@{1} = forrige. HEAD@{2} = før den igjen 
ORIG_HEAD = Forrige HEAD. ORIG = før prøve innsjekk. 

refs/heads = lokale branches
refs/remotes = repo branches
refs/remotes/origin/master = master branchen i repo

master = siste commit (remote = origin/master (origin sin master))

Check out nyeste:
git checkout master = sjekker ut origin/master
