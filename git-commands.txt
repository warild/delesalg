git fetch = henter nyeste remote/innsjekk  

git merge = slå sammen remote/master med lokal/master på aktiv branch
git merge branchX  = merge annen branch med den som er lastet nå

git pull = oppdater lokalt utsjekk med nyeste remote/innsjekk  
= git fetch + git merge

--- Checkout nyeste innsjekk på en branch (eller på remote/master)
git checkout testBranch
git checkout master

--- Opprett branch
git branch 
git branch  testBranch
git branch -d branchX   slett brandh (peker)

git push <branch>

git log = lister de siste commits

git reflog = se liste med alle commits

Stash - legge til side lokalt som stakk.
git stash
git stash pop 
git stash list



Branch peker på en bestemt commit
Master peker på master -> HEAD

HEAD = peker til aktuell lokal commit
FETCH_HEAD = peker til remote siste commit

fetch + merge = pull
